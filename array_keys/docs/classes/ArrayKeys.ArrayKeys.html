my_array_keys()

my_array_keys(array  $array = array(), mixed  $searchValue = null, boolean  $strict = false, boolean  $expandArray = false, integer  $depth, integer  $MAX_DEPTH = INF, array  $keys = array()) : array

Enhanced clone of the PHP standard function 'array_keys()'.

array_keys() returns the keys, numeric and string, from the array. If the optional search_value is specified, then only the keys for that value are returned. Otherwise, all the keys from the array are returned.

This function can also iterate over the input array recursively to retrieve even all in-depth keys.

Parameters

array $array

The input array to search keys.
mixed $searchValue

(Optional) If specified, then only keys containing these values are returned. It is null by default, which means that will return all indexes.
boolean $strict

(Optional) Determines if strict comparison (===) should be used during the search. It is false by default.
boolean $expandArray

(Optional) Only if this parameter is set to true, the function will act recursively. You may customize the next two parameters. Its default value is false.
integer $depth

(Optional) Current input array's depth level. Its default value is 0 (top level of $array).
integer $MAX_DEPTH

(Optional) Maximum depth iteration. Its default value is INF (bottom level of $array).
array $keys

(Optional) An array containing keys to return. You can add the returning array to an existing one. Simply invoke this function like this "$your_array[] = my_array_keys(($array, $searchValue, $strict, $expandArray, $depth, $MAX_DEPTH, $your_array);".

Throws

\Exception

General standard Exception exceptions with descriptive messages of the error.

Returns

array —

Containing keys.
